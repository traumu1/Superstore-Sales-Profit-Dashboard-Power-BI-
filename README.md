# Power BI Superstore Dashboard

## Description
This dashboard visualizes the Kaggle Superstore dataset in Power BI and connects directly to **Azure Blob Storage (Free Tier)**.

## Visualizations
- **KPIs:** Total Profit, Total Sales, Total Orders  
- **Profit by Category**  
- **Sales/Profit by Region and State**

## Data Source
- Azure Blob Storage (Free Tier)  
- CSV file: `Superstore.csv`

## Notes
- Dashboard file included as `.pbix`
